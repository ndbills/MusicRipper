# MusicRipper Logo Concepts

This folder keeps the exploratory logo and README hero assets so the design trail is not lost.

## Current Picks

- App icon / mark: `prompt-d-audiophile-disc-badge/prompt-d-variant-01.svg`
- README hero favorite: `readme-hero-logo-concepts/readme-hero-concept-14.svg`
- Exported app hero assets: `../musicripper-hero.svg` and `../musicripper-hero.png`

## Design Trail

- Prompts A-C explored generic app-icon directions.
- Prompt D established the final app-logo direction: a flat-vector audiophile disc mark with a chunky charcoal puck, full cyan/violet/amber edge treatment, cream center label, tan rings, crimson center dot, and soft shadow.
- Prompt D variant 01 became the canonical source mark. Later variants and lockups keep that mark's geometry/colors intact and only transform/place it.
- Prompts E-J2 explored wordmarks, horizontal lockups, stacked README heroes, disc-as-tittle ideas, and narrative banners. These are retained for reference but were not selected.
- README hero concepts 01-16 explored GitHub README banner compositions using the canonical disc mark.
- Concepts 08, 11, and 12 converged on a centered masthead with horizontal tan rules.
- Concept 14 is the current favorite: it combines concept 12's spacing/rules with concept 11's tagline, `Secure CD ripping for a bit-perfect FLAC library`, and gives the disc enough presence without overlapping the wordmark.

## Future Generations

Place any new logo or hero explorations under `assets/logo-concepts/`, not `docs/images/`.
